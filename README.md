# 📝 Simple To-Do List App

A full-stack **To-Do List application** built with **Node.js, Express, MongoDB, and Vanilla JavaScript**.  
Organized into `frontend` and `backend` folders for clarity and scalability.

---

## 🚀 Features
- Add, update, and delete tasks
- Mark tasks as done/undone
- Tasks are stored in **MongoDB**
- Simple, responsive UI with HTML, CSS, and JavaScript
- Separate backend and frontend for easy development

---

## 🛠️ Technologies
- **Backend:** Node.js, Express, MongoDB, Mongoose, CORS  
- **Frontend:** HTML, CSS, JavaScript (Vanilla)  
- **Optional Dev Tool:** Nodemon (for auto-restart during development)
